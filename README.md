# Civil 3D サブアセンブリビューア

Autodesk Civil 3D のサブアセンブリのコードを視覚的に確認できるWebツールです。

🔗 **デモサイト**: https://yuri-haga.github.io/civil3d-subassembly-viewer/

## 特徴

- **視覚的な確認**: SVG図面上でサブアセンブリの構造を確認
- **詳細情報の表示**: ポイント（P1、L1等）にマウスオーバーでコード情報をツールチップ表示
- **コードのコピー**: ワンクリックでコードをクリップボードにコピー
- **複数のサブアセンブリ**: 側溝、路肩、法面展開など複数のサブアセンブリに対応

## 使い方

1. デモサイトにアクセス
2. 左上のメニュー（≡）から見たいサブアセンブリを選択
3. SVG図面上のP1、L1などにマウスを当てるとツールチップが表示されます
4. クリックすると右側パネルに詳細情報が固定表示されます
5. 「コピー」ボタンでコードをクリップボードにコピーできます

## 対応サブアセンブリ

- 側溝
- 内側横断勾配摺り付け車線
- 小段付き法面展開（盛土）
- 小段付き法面展開（切土）
- 路肩（路盤垂直）
- 切土法面_水路
